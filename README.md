# Planilha de Vendas Mensal

Bem-vindo ao Planilha de Vendas Mensal, um site dedicado ao estudo e desenvolvimento de planilhas e gráficos usando HTML, CSS e JavaScript.

## Sobre

Planilha de Vendas Mensal é um projeto em desenvolvimento onde estou criando planilhas e gráficos como uma forma de aprofundar meus conhecimentos em HTML, CSS e JavaScript. O site apresentará uma variedade de planilhas e gráficos interativos focados em vendas mensais.

## Recursos

- Planilhas interativas criadas com HTML e CSS
- Gráficos dinâmicos gerados com JavaScript
- Estudos detalhados sobre o desenvolvimento de planilhas e gráficos
